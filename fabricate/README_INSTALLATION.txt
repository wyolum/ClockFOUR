To create and modify mechanical components for ClockFOUR/Chronogram please install the following pieces of software:

Install Python 2.7x
https://www.python.org/downloads/release/python-279/

PIL for Python 2.7
http://www.pythonware.com/products/pil/

Reportlab for Python 2.7
https://bitbucket.org/rptlab/reportlab

Numpy for Python 2.7
http://sourceforge.net/projects/numpy/files/NumPy/1.9.1/
